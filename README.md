# Byte API
## Steps to Run DEV

1. Clonar proyecto
2. ```yarn install```
3. Cambiar las variables de entorno, ajustar ```.env``` si es necesario
5. Levantar la base de datos
```
docker-compose up -d
```
6. Levantar ```npm run start:dev```
